# 1-my_list.txt

===============================
 How to Use 1-my_list.py
===============================

This module has one funtion called ''print_sorted()''

Usage
=====

''print_sorted()'' prints sorted order from the inherited class

::

	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> my_list.append(2)
	>>> my_list.append(1)
	>>> my_list.append(3)
	>>> my_list.append(4)
	>>> print(my_list)
	[2, 1, 3, 4]
	>>> my_list.print_sorted()
	[1, 2, 3, 4]
	>>> print(my_list)
	[2, 1, 3, 4]
